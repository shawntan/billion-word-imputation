Logistic Regression / Perceptron / Single-layer Neural Network

For some binary classification task, we need to fit some weights $w$ such that $$f(X) = \left\{\begin{matrix} 1 & \text{if } w^\top X > 0 \\ 0 & \text{else} \end{matrix}\right.$$ Differentiable alternative, logistic function: $$\sigma(x) = \frac{1}{1-e^{-x}}$$ To optimise, log-loss function: $$E(X,y) = - \frac{1}{n} \sum_{i=1}^n \left[ y_i \log(f(X_i)) + (1 - y_i) \log(1 - f(X_i))\right]$$

Backpropagation

Backpropagation is gradient descent performed on the entire neural network function.

Since the neural network is composed of other neural networks, backprop is applying differentiation using the chain-rule recursively.

Language Models using Neural Networks¶

Language model:

Given a context of $k$ previous words, what's the probability of the next word, $$P(w_t|w_{t-1},w_{t-2},\dots,w_{t-k})$$

We can create a neural network that takes in the previous $k$ words, and outputs the probabilities of all $|V|$ words

Bengio, Yoshua, et al. "Neural probabilistic language models." Innovations in Machine Learning. Springer Berlin Heidelberg, 2006. 137-186.
